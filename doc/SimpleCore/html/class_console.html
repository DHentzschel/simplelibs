Prints the specified string to console with new line depending on newLine parameter.

If set, the parameters color and backgroundColor could be changed. Default color is LightGrey, default backgroundColor is Black. If parameter centered is set to true, the console prints the string centered in the console.

Parameters
stringthe string to print
newLinewhether a new line should be appended to console
colorthe foreground color of the string to print
backgroundColorthe background color of the string to print
centeredwhether the string should be printed centered
